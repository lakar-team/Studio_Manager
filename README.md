The next evolution of your project control center. A premium, ultra-lightweight Node.js engine for managing agentic workflows, tracking code evolution with a "Time Machine" interface, and providing high-fidelity context for AI Agents.

## ✨ Features

- 💎 **Premium Dark Mode UI**: A high-tech dashboard built with Tailwind CSS, Inter typography, and Lucide icons.
- 🕒 **Time Machine**: Seamless Git integration to visualize commit history, create snapshots, and restore versions with one click.
- 🧠 **AI Neural Context**: One-click generation of full-project awareness payloads. Skip the manual copy-pasting and give your AI agents "Perfect Situational Awareness."
- 📝 **AI Persistent Instructions**: A dedicated space for storing long-term context and instructions in `AI_Instructions.md`, ensuring your agents never lose track of project goals.
- 🚀 **Zero-Config Ignition**: A self-contained Node.js server with no external dependencies—just run and go.

## 🛠️ Quick Start

### 1. Prerequisites
Ensure you have **Node.js 18+** and **Git** installed on your system.

### 2. Ignition
Navigate to the root and blast off using the launcher:
```powershell
.\LocalGitRepo\launch.bat
```
Alternatively, run the server manually:
```bash
cd LocalGitRepo
node server.js
```
The studio will be available at `http://localhost:8081`.

## 📦 Project Structure

- `LocalGitRepo/server.js`: The ultra-lightweight Node.js engine.
- `LocalGitRepo/launch.bat`: The one-click startup script.
- `AI_Instructions.md`: Your AI agent's long-term memory.

## 🤖 AI Agent Workflow

Lakar Studio Manager is designed as a bridge between the local filesystem and AI Coding Agents.
1. Use the **"Generate AI Context"** button to compile your entire project structure and source code.
2. Use the **"AI Instructions"** panel to define core logic and persistent project rules.
3. Paste the generated context into any LLM to give it an instant 100% "Perfect View" of your current codebase.
