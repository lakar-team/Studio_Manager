# 🛸 LAKAR STUDIO MANAGER v2.0

A premium, lightweight control center for managing local Agency projects, tracking code evolution with a "Time Machine" interface, and providing high-fidelity context for AI Agents.

## ✨ Features

- 💎 **Premium UI**: High-tech dashboard built with Tailwind CSS and Lucide icons.
- 🕒 **Time Machine**: Built-in Git integration for creating snapshots and restoring previous versions with one click.
- 🧠 **AI Neural Context**: One-click generation of full-project awareness payloads for Gemini, Claude, or GPT.
- 📁 **Project Discovery**: Easily link multiple local repositories to a single unified console.
- 🚀 **Lightweight**: Zero-dependency frontend (CDN-based) and a simple Flask backend.

## 🛠️ Quick Start

### 1. Prerequisites
Ensure you have **Python 3.10+** and **Git** installed on your system.

### 2. Installation
Clone the repository and install the requirements:
```bash
git clone https://github.com/lakar-team/Studio_Manager.git
cd Studio_Manager
pip install -r requirements.txt
```

### 3. Ignition
Run the console:
```bash
python console.py
```
Open your browser to `http://localhost:8000`.

## 📦 Project Structure

- `console.py`: The core Flask engine and UI logic.
- `projects.json`: Configuration storage for your linked repositories.
- `requirements.txt`: Python dependencies.

## 🤖 AI Agent Workflow

Lakar Studio is designed to work in tandem with AI Coding Agents. Use the **"AI Context"** button to generate a structured dump of your entire project. Paste this into your LLM of choice to give it "Perfect Situational Awareness" of your codebase instantly.
